# 📂 DocuVault — Digital Document Manager

DocuVault is a modern and responsive Digital Document Management System built using React.js, Redux Toolkit, and Cloudinary. It allows users to upload, view, search, and manage digital files such as images and documents with a clean UI and real-time state management.

The application uses Redux Toolkit for centralized state handling and Cloudinary for cloud-based file storage, ensuring fast and efficient file operations.

## 🚀 Features

- Upload documents (images, PDFs, and other files)
- View uploaded files in a structured card layout
- Search files by name
- Delete files instantly with UI update
- Cloud storage integration using Cloudinary
- Real-time state management using Redux Toolkit

## 🛠 Tech Stack

Frontend: React.js (Vite)  
State Management: Redux Toolkit  
Async Handling: Redux Thunk  
Cloud Storage: Cloudinary  
Styling: Custom CSS (Premium Eye-Comfort Brown Theme)
